# 🎥 MeetLink

MeetLink is a full-stack **real-time video conferencing platform** built with the MERN stack. It allows users to host and join live meetings with support for video/audio calls, screen sharing, and in-session chat — all wrapped in a clean, modern UI.

🔗 **Live Demo:** [https://meet-link-pi.vercel.app/](https://meet-link-pi.vercel.app/)

---

## ✨ Features

- 🔐 **Authentication & Authorization** — Signup, Login, and Forgot Password (JWT-based)
- 🎦 **Host & Join Sessions** — Create a session as a host or join via a shared link
- 👥 **Live Participant Tracking** — Hosts can view real-time participant count
- 🛑 **Host Controls** — Host can end the session at any time
- 💬 **In-Session Chat** — Real-time messaging during meetings
- 🖥️ **Screen Sharing** — Share your screen with other participants
- 🎙️ **Audio/Video Calls** — Powered by ZegoCloud SDK
- 📊 **Session Dashboard** — View All, Active, and Ended sessions with creation/end time
- 🎨 **Modern, Responsive UI** — Styled using Tailwind CSS

---

## 🛠️ Tech Stack

**Frontend:** React.js, Tailwind CSS
**Backend:** Node.js, Express.js
**Database:** MongoDB
**Real-Time Communication:** ZegoCloud SDK
**Authentication:** JWT
